개인정보처리방침

채아피싱 (낚시 달력) · 최종 업데이트: 2026-06-21

채아피싱은 사진 원본과 조과 기록을 자체 서버에 저장하지 않습니다. 사진 분석 결과, EXIF 정보, 위치 기반 기록, 사용자 수정 데이터는 기기 내부에 저장됩니다. 날씨와 물때 조회를 위해 촬영 날짜와 대략적인 위치가 외부 API로 전송될 수 있습니다. 사용자는 설정에서 로컬 데이터와 백업 파일을 삭제할 수 있습니다.

1. 접근하는 데이터와 목적

사진 및 동영상, 사진 EXIF의 위치 정보(GPS), 앱 설정/백업/알림 상태에 접근합니다. 갤러리에서 낚시 사진을 찾고 날짜별로 정리하기 위한 목적입니다.

2. 로컬 저장 원칙

사진 원본은 외부 서버에 업로드하지 않으며, 분석 결과/조과 기록/사용자 보정값은 기기 내부 SQLite(Room)에 저장됩니다.

3. 외부 API 전송 데이터

과거 날씨/물때 조회를 켠 경우 날짜, 시간, 대략적인 좌표(정밀도 낮춤 가능)가 외부 API로 전송될 수 있습니다. 사진 원본/썸네일/전체 조과 기록은 전송하지 않습니다.

4. Android Auto Backup

Android 백업 기능이 켜져 있으면 앱 데이터(DB·설정)가 사용자의 Google 계정 백업 공간에 백업될 수 있습니다. 사진 원본/썸네일은 제외됩니다.

5. 수동 백업 파일

사용자가 내보낸 .jogwa.zip 백업 파일은 사용자가 직접 관리하며, 사진 원본은 포함되지 않습니다.

6. 데이터 삭제 및 권한 철회

설정 > 데이터 관리에서 분석 결과/전체 데이터를 삭제할 수 있고, 시스템 설정에서 사진/알림 권한을 철회할 수 있습니다.

7. 보관 기간 및 제3자 제공

데이터는 사용자가 삭제하기 전까지 기기에 보관되며, 제3자에게 제공·판매하지 않습니다. 광고/트래킹 SDK를 포함하지 않습니다.
