이용 가이드

채아피싱 (낚시 달력) · 최종 업데이트: 2026-07-08

채아피싱은 갤러리 속 낚시 사진을 자동으로 찾아 날짜별로 정리해 주는 ‘낚시 달력’ 앱입니다. 사진 분석은 모두 기기 안에서 이루어지며, 사진 원본은 외부로 전송되지 않습니다.

1. 처음 시작하기

  • 첫 실행 시 사진 접근 권한을 허용하면 갤러리에서 낚시 사진을 찾기 시작합니다.
  • 첫 분석은 사진 수에 따라 시간이 걸릴 수 있습니다. 진행 알림에서 ‘중지’로 멈출 수 있고, 다음에 이어서 분석됩니다.
  • Android 13 이상에서는 알림 권한을 허용해야 분석 완료·백업 알림을 받을 수 있습니다.

2. 달력 탭

  • 날짜별 낚시 사진 컷수를 한눈에 볼 수 있습니다.
  • 월 요약은 ‘낚시 사진 N컷 · 영상 M컷’, 바다·민물 컷수, 낚시한 날수를 보여줍니다.
  • 월 요약에 ‘? 미정’이 보이면 아직 바다/민물이 정해지지 않은 컷이 있다는 뜻입니다. 타일을 누르면 설명이 뜨고, 사진을 열어 [바다]/[민물] 버튼으로 직접 지정하면 통계에 반영됩니다.
  • 그 아래 ‘이 달의 패턴’은 자주 나간 물때·시간대를 정리해 줍니다. 물때는 바다 낚시 개념이라, 민물만 있는 달에는 물때 대신 시간대만 표시됩니다.
  • 날짜를 누르면 그날의 상세(요약글·날씨/물때·사진)를 볼 수 있습니다.

3. 사진 탭 (조과컷)

  • 좌우로 밀어 4가지 보기를 전환합니다: 낚시 사진 / 확인 필요 / 위치 있는 사진만 / 사진 없음.
  • 확인 필요는 낚시인지 애매해 판단을 보류한 사진이 모이는 곳입니다. ‘대기/제외됨’으로 나뉘어 있어요 — 사진을 열어 [낚시]로 올리거나 [제외]로 정리하면 대기 목록에서 사라지고, 실수로 제외한 사진은 ‘제외됨’에서 다시 [낚시]로 복구할 수 있습니다.
  • 확인 필요에서 [낚시]로 올리면 사진 성격에 맞는 자리(조과컷 또는 포인트)로 들어갑니다. 풍경 사진 하나를 낚시로 확정하면, 같은 날 함께 보류됐던 풍경 사진들도 자동으로 복귀합니다.
  • 사진 없음은 낚시 사진으로 기록됐던 원본이 갤러리에서 삭제·이동돼 찾을 수 없게 된 사진이 모이는 곳입니다. 원본이 사라져도 날짜와 기록은 달력에 남습니다.
  • ‘위치 있는 사진만’은 지도로 표시됩니다. 지도 오른쪽 아래의 슬라이드 스위치를 끄면 지도를 자유롭게 이동(패닝)할 수 있고, 켜면 좌우 스와이프로 탭을 넘깁니다.
  • 사진을 누르면 전체 화면 뷰어로 열리며, 같은 보기의 사진들을 넘겨볼 수 있습니다. 뷰어 하단 버튼으로 수질([바다]/[민물]) 지정, [낚시] 승격, [제외]를 바로 처리할 수 있어요.
  • 목록을 스크롤하는 동안 오른쪽에 맨 위/맨 아래로 바로 가는 버튼이 잠깐 나타납니다.
  • 분석이 진행 중일 때는 상단에 ‘사진 분석 중’ 배너가 표시됩니다. 분석 중에 보이는 목록은 중간 상태이며, 분석이 끝나면 사진이 자동으로 다시 정리됩니다.

4. 날짜 상세

  • 자동 생성된 요약글과 그날의 날씨·물때 정보를 카드로 보여줍니다. 날씨는 없으면 자동으로 한 번 조회하고(설정에 따름), 물때는 인터넷 없이 기기 안에서 계산됩니다.
  • ‘출조 편집’ 한 화면에서 정리할 수 있습니다. ‘사진 정리’ 탭에서 여러 사진의 분류를 한 번에 바꾸고, ‘어종·마릿수’ 탭에서 실제 잡은 어종·마릿수를 여러 종 이어서 기록합니다.
  • 자동 분석이 놓친 날은 ‘이 날 조과 추가’로 그날 갤러리에서 사진을 직접 골라 낚시 사진으로 등록할 수 있습니다.

5. 통계 탭

  • 전체 낚시 사진·영상 수, 작년 같은 달 기록을 요약해서 보여줍니다.
  • ‘연도별 낚시 사진’은 연도를 펼쳐 월별 막대 차트로 보거나, 오른쪽 ‘차트/월별’ 토글로 월별 목록 보기로 전환할 수 있습니다. 월별 목록에는 컷수와 함께 그달 낚시한 날수가 표시되고, 막대나 월을 누르면 달력의 해당 월로 이동합니다.
  • ‘패턴 분석’은 물때·시간대·계절별 컷수와 골든 컨디션(자주 나간 조건·잘 잡힌 조건)을 보여주며, 바다/민물로 나눠 볼 수 있습니다.

6. PC로 전송

  • 설정 > PC로 전송에서, 같은 네트워크(같은 공유기)에 있는 PC의 공유 폴더로 낚시 사진·영상을 복사할 수 있습니다. PC는 Wi-Fi든 랜선이든 상관없습니다.
  • PC 주소를 직접 입력하거나 ‘같은 네트워크 PC 찾기’로 찾을 수 있습니다(Windows는 자동 검색에 안 잡힐 수 있어 IP를 직접 입력).
  • 사진은 ‘기준 폴더/년/월’ 구조로 정리되고, 이미 보낸 파일은 건너뜁니다. 전송할 때 최신 백업 파일(.fishing.zip)도 기준 폴더에 함께 복사됩니다.
  • 모든 전송은 사용자가 직접 시작할 때만 이루어지며, 사용자 소유의 PC로만 복사됩니다(외부 서버로 보내지 않습니다).

7. 설정

  • 환경설정: 알림 설정, 사진 분석 설정(자동 분석), 환경 데이터 설정(지도 엔진 선택, 날씨 조회·좌표 정밀도, 데이터 전송 안내), 백업 및 복원, PC로 전송, 데이터 관리(재검사·재분석·초기화).
  • 물때는 인터넷 없이 기기 안에서 계산되므로 켜고 끄는 설정이 없으며 항상 표시됩니다. 좌표 정밀도 낮추기는 날씨 조회 시 위치를 대략적으로만 보내도록 하는 옵션입니다.
  • 백업은 사진 원본을 제외한 기록·설정만 .fishing.zip 파일로 내보내고 가져올 수 있습니다.
  • 데이터 관리의 각 항목은 누르면 어떤 작업인지 설명이 먼저 뜨고, ‘시작’을 눌러야 실행됩니다.
  • 사진 재검사는 갤러리에서 삭제·이동된 사진을 찾아 ‘사진 없음’으로 표시하고, 다시 생긴 사진은 복구합니다.
  • 앱 업데이트로 분류 방식이 개선됐을 때는 ‘빠른 재처리’(초 단위)로 기존 분석 결과에 새 규칙만 다시 적용할 수 있습니다. 분석 자체를 다시 하려면 ‘전체 재분석’을 사용하세요(사진이 많으면 오래 걸립니다). 어느 쪽이든 사용자가 직접 수정한 값은 유지됩니다.

8. 사진 분석에 대하여

  • 낚시 사진 판별은 기기 내 인공지능(온디바이스) 모델의 추정 결과로, 100% 정확하지 않습니다. 잘못 분류된 사진은 사진 탭이나 일괄 수정에서 직접 바로잡을 수 있습니다.
  • 해변·노을 같은 풍경 사진은 진짜 낚시 포인트 사진과 겉모습만으로 구분하기 어렵습니다. 그래서 풍경 사진은 같은 날 낚시 흔적(물고기·장비 사진, 직접 기록한 조과)이 있을 때만 낚시 사진으로 올라가고, 없으면 ‘확인 필요’에 보류됩니다. 보류된 사진 중 하나를 [낚시]로 확정하면 그날의 나머지 풍경 사진도 자동으로 복귀합니다.
  • 사용자가 직접 바로잡은 분류(제외/낚시 지정)는 재분석·재처리가 절대 되돌리지 않습니다.
  • 날씨·물때 정보도 추정값이며 실제와 다를 수 있습니다.

9. 정보의 출처와 정확도

  • 날씨: Open-Meteo 공개 API로 조회한 값입니다. 조회 시 촬영 날짜와 대략적인 좌표만 전달하며, 좌표 정밀도 낮추기를 켜면 위치를 더 대략적으로 보냅니다. 예보·재분석 값이라 실제 현장과 다를 수 있습니다.
  • 물때: 국립해양조사원 관측에서 뽑은 조화상수를 기기 안에 내장해 인터넷 없이 계산합니다. 지점·날짜에 따라 실제 조석과 오차가 있을 수 있고, 표준 관측소에서 먼 곳일수록 오차가 커질 수 있습니다.
  • 지도: 설정에서 고른 지도 엔진(카카오맵 또는 구글맵)으로 표시합니다. 지도를 열 때만 좌표가 해당 지도 서비스로 전달됩니다.
  • 지명: 좌표를 안드로이드 시스템 지오코더로 주소로 바꿔 요약에 넣습니다. 바다·외딴 곳은 가까운 지명으로 대체되거나 빈 값일 수 있습니다.
  • 이 값들은 참고용이며, 잘못된 부분은 사진을 열어 직접 바로잡을 수 있습니다.

10. 개인정보

사진 원본과 조과 기록은 외부 서버에 저장하지 않으며 기기 안에만 보관됩니다. PC로 전송 기능은 사용자가 직접 지정한 같은 네트워크의 PC로만 복사합니다. 자세한 내용은 개인정보처리방침을 확인하세요.
