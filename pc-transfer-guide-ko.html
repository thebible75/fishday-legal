PC로 전송 — 공유 폴더 준비 안내

채아피싱 · 폰 사진·영상을 같은 Wi-Fi 의 PC 로 복사 · 최종 업데이트: 2026-07-10

이 방식은 PC 에 공유 폴더를 하나 만들고, 앱이 그 폴더로 사진을 SMB 로 복사합니다. 폰과 PC 가 같은 공유기(같은 네트워크)에 있으면 유선·무선 상관없습니다. 아래에서 내 PC 에 맞는 탭을 골라 ① 공유 폴더 설정 → ② IP·계정 확인 → ③ 앱에 입력 순서로 준비하세요.

더 간편한 방법을 원하시면, 공유 폴더·계정 설정이 필요 없는 « QR 로 PC 연결 » 방식도 있습니다(앱 설정에서 선택). 이 문서는 기존 SMB 공유 폴더 방식 전용입니다.

준비 방법

1. 공유 폴더 설정

  1. 사진을 저장할 폴더를 우클릭 → 속성 → [공유] 탭.
  2. [공유] 버튼 → 내 계정을 추가하고 사용 권한을 «읽기/쓰기» 로 바꿉니다.
  3. [고급 공유] → «이 폴더 공유» 체크. 여기 표시된 공유 이름을 앱의 공유 이름 칸에 그대로 입력합니다.
  4. 제어판 → 네트워크 및 공유 센터 → 고급 공유 설정 에서 «파일 및 프린터 공유» 를 켭니다.
  5. 앱 계정 = Windows 로그인 계정 + 비밀번호.
⚠ 비밀번호가 없는 계정은 연결이 거부됩니다. 계정에 비밀번호를 설정하세요.

2. IP 주소 찾기

시작 → cmd 실행 → ipconfig 입력 → «IPv4 주소» (예: 192.168.0.10).

3. 계정 이름 찾기

시작 → cmd 실행 → whoami 입력 → «PC이름\계정이름» 에서 \ 뒤가 사용자 이름입니다.

1. 공유 폴더 설정

  1. 시스템 설정 → 일반 → 공유 에서 «파일 공유» 를 켭니다.
  2. [공유 폴더] 에 원하는 폴더를 추가합니다. 그 폴더 이름이 «공유 이름» 입니다.
  3. [옵션…] → «SMB를 사용하여 파일과 폴더 공유» 체크 → 아래에서 내 계정을 체크하고 비밀번호를 입력합니다.
  4. 앱 계정 = Mac 계정의 «단축 이름» + 비밀번호.

2. IP 주소 찾기

시스템 설정 → Wi-Fi → [세부사항…] → «IP 주소». 또는 앱 주소 칸에 컴퓨터이름.local 을 사용해도 됩니다.

앱에서 연결하기

  1. 앱 설정 → PC로 전송 을 엽니다.
  2. PC 주소(IP 또는 이름.local), 공유 이름, 계정·비밀번호 를 입력합니다. (같은 네트워크의 PC 는 «자동 찾기» 로 주소를 채울 수도 있습니다.)
  3. 연결 확인 후 전송 을 누르면 사진이 복사됩니다. 다음부터는 새 사진만 보냅니다(증분).

사진은 공유 폴더 안에 FishingDay/년/월 로 자동 정리됩니다.

연결이 안 될 때

  • 폰과 PC 가 같은 공유기에 있는지 확인하세요.
  • PC 방화벽에서 파일 공유(SMB) 를 허용했는지 확인하세요.
  • 공유기의 «AP 격리 / 기기 간 통신 차단» 이 켜져 있으면 꺼주세요.
  • 주소·공유 이름·계정 을 다시 확인하세요(대소문자 구분).
  • Windows 는 계정에 비밀번호가 있어야 하며, 없으면 연결이 거부됩니다.

안전

전송은 사용자 소유의 PC 로만 이루어지며, 개발자·제3자 서버로는 어떤 데이터도 가지 않습니다. 연결과 사진은 같은 네트워크(로컬) 안에서만 오갑니다. PC 접속 정보는 기기 내부에만 저장됩니다.
